Politique de confidentialité — Strumetric

Dernière mise à jour : 18/04/2026

Cette Politique de confidentialité explique comment l’application Strumetric traite les données.

Important : ce document est un modèle à relire/adapter selon les SDK utilisés (analytics, pubs, crash reporting), les données réellement collectées et les obligations légales applicables.

1. Responsable de traitement

Responsable : Tutoguitare France / Tutio Studio
Contact : tutoguitare.france@gmail.com

2. Données traitées

Par défaut, Strumetric peut fonctionner sans création de compte. Selon la configuration de l’Application, les catégories suivantes peuvent être traitées :

  • Données d’usage (ex : fonctionnalités utilisées, écrans visités) — si analytics activé.
  • Données techniques (ex : modèle appareil, version OS, logs) — notamment en cas de crash.
  • Données locales (ex : rythmes/mémoires) stockées sur ton appareil via stockage local.

3. Finalités

  • Fournir et améliorer le service.
  • Diagnostiquer les bugs et améliorer la stabilité.
  • Mesurer l’usage (si activé).

4. Base légale

  • Exécution du contrat : fournir l’Application.
  • Intérêt légitime : sécurité, prévention de fraude, amélioration produit.
  • Consentement : si un module nécessite un consentement (ex : analytics/publicité), il sera demandé.

5. Partage des données

Strumetric ne vend pas tes données et ne les partage pas avec des tiers à des fins publicitaires. Les données de configuration (rythmes, mémoires) sont stockées localement sur ton appareil..

6. Stockage et durée

  • Données locales : restent sur ton appareil sauf action de ta part (suppression app / reset).
  • Logs / diagnostics : conservés pendant une durée proportionnée : 90 jours.

7. Tes droits

Selon la réglementation applicable (RGPD, etc.), tu peux demander l’accès, la rectification, l’effacement, la limitation ou l’opposition. Contact : tutoguitare.france@gmail.com.

8. Sécurité

Des mesures raisonnables sont mises en œuvre pour protéger les données. Aucune méthode n’est sûre à 100%.

9. Enfants

L’Application n’est pas destinée aux enfants de moins de 4 ans sans accord parental, si applicable.

10. Modifications

Cette politique peut être mise à jour. La version applicable est celle publiée sur cette page.
